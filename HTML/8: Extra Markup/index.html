Escape Characters Example

Here are some common escape characters in HTML:

  • < - less-than sign
  • > - greater-than sign
  • & - ampersand
  • " - double quote
  • ' - single quote
  •   - non-breaking space

If you want to include one of these characters in your HTML, you can use the corresponding escape sequence instead. For example:

This is a <b>bold</b> text.

This is an “example” sentence.

This is a “quote” mark.

This is a &euro; symbol.
